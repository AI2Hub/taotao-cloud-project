```
Spring Cloud Alibaba 2.2.5.RELEASE 、
Spring Boot websocket 、


前台基于
react 、
tarojs 、
sa 、
typescript 、
viet 、
postcss 、
tailwindcss 、

```

#### 4.安装教程

``` bash
# 安装依赖
npm install 或者 yarn 

# 启动服务 localhost:3000
taotao-cloud-mall:dev:weapp

# 构建生产环境
taotao-cloud-mall:build:weapp
```
